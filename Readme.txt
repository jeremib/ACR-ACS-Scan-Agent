README.TXT

Solution consists of two projects:

1.AcsAcr122UScanAgent (windows application)
2.Acr122uScanAgentInstaller (InstallShield- used to create installer for 1.AcsAcr122UScanAgent  application)

To create new installer you need to build Acr122uScanAgentInstaller
and go to Acr122uScanAgentInstaller/Acr122uScanAgentInstaller/Express/DVD-5/DiskImages/DISK1
DISK1 folder will contain all needed files for new installer.


TO install application just open setup.exe and complete all steps step by step.

Please see attached html file WebSocketServerConsumerExample.html for information about how to 
communicate with websocketserver from browser using java script.
